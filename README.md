# empathic-echo

# Empatik Yankı

Bu depo, empati temelli dijital etkileşimler üzerine geliştirilecek
kavramsal ve deneysel bir proje için oluşturulmuştur.

Bu aşamada içerik taslak ve geliştirme amaçlıdır.
Bu depo, dijital ortamlarda psikolojik zarar, toksik dil ve gizli mobbing riskini
tespit etmeye yönelik bir analiz servisinin taslak projesidir.

Kod paylaşımı değil, hizmet modeli hedeflenmektedir.
